# QrApplication Landing Page

Landing page para QrApplication - la app definitiva de códigos QR para Android.

## Características

- Escaneo instantáneo con tecnología CameraX
- Privacidad total - datos guardados localmente
- Sin publicidad
- Modo offline
- Diseño moderno con Tailwind CSS y Radix UI

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 4
- **UI Components**: Radix UI + shadcn/ui
- **Animations**: Framer Motion
- **Icons**: Radix UI Icons + Lucide React

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production (static export)
npm run build
```

## Deploy a GitHub Pages

El proyecto está configurado con GitHub Actions para deploy automático:

1. Habilita GitHub Pages en tu repositorio:
   - Settings > Pages > Source: **GitHub Actions**

2. Haz push a la rama `main` y el workflow se ejecutará automáticamente

3. La URL será: `https://tu-usuario.github.io/Qr-app-lp/`
